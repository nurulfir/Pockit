## **Kelola Keuangan Anda, Raih Kesejahteraan Finansial.**

Pockit adalah aplikasi pencatatan keuangan yang intuitif dan mudah digunakan. Dirancang dengan filosofi "pocket it" (masukkan ke saku), aplikasi ini membantu Anda memegang kendali penuh atas setiap rupiah pemasukan dan pengeluaran Anda, memastikan Anda selalu *aware* dengan kondisi finansial Anda.

-----

### ğŸŒŸ Fitur Utama (Key Features)

Pockit hadir dengan serangkaian fitur yang dirancang untuk mempermudah manajemen uang harian Anda:

  * **Pencatatan Transaksi Cepat:** Catat setiap **Pemasukan** dan **Pengeluaran** dalam hitungan detik.
  * **Target Tabungan (*Savings*):** Buat target tabungan spesifik dan pantau progres pencapaian Anda.
  * **Anggaran Belanja (*Spending Limits*):** Tetapkan batas pengeluaran bulanan atau mingguan pada kategori tertentu untuk mencegah pengeluaran berlebihan.
  * **Pengingat Tagihan (*Bills Reminder*):** Jangan pernah telat membayar tagihan lagi\! Dapatkan notifikasi untuk tenggat pembayaran yang akan datang.

### ğŸš€ Instalasi & Penggunaan

Ikuti langkah-langkah sederhana di bawah ini untuk menjalankan Pockit di lingkungan lokal Anda.

#### 1\. Kloning Repositori

Buka terminal Anda dan klon proyek menggunakan URL GitHub:

```bash
git clone https://github.com/nurulfir/Pockit.git
```

#### 2\. Instal Dependensi

Masuk ke dalam direktori proyek dan instal semua paket yang diperlukan:

```bash
cd Pockit
npm install
```

#### 3\. Menjalankan Aplikasi

Jalankan aplikasi dalam mode pengembangan. Aplikasi akan terbuka di *browser* Anda (biasanya di `http://localhost:5173`):

```bash
npm run dev
```
